# 入出力ファイル一括取得プログラム
## 概要　
このプログラムは二段階で実行する。まず、ワークフローIDを指定してこのワークフローを実行したランの情報を全て取得する。その後その情報からラン毎の入出力ファイル内容を取得する。
必要なものは以下となる。

* ワークフローID
* APIトークンまたはログイン情報
* siteID
* 入出力情報の変換対応テーブル

## システム構成

本スクリプトを実行するために必要なシステム構成を記述する。

* OS
  + python3.6が実行できれば特定のOSに縛られない。
* python
  + Version3.6以降
  + requests package
  + misrc_workflow_python_lib(from gitlab)
